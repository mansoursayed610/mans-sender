# 📱 MANS SENDER v2.0

نظام إرسال حملات واتساب الاحترافي

## التشغيل المحلي
```bash
npm install
npm start
```
افتح المتصفح على: http://localhost:3001

## الرفع على Render
1. ارفع المشروع على GitHub
2. أنشئ Web Service على render.com
3. Start Command: `node server.js`

## بيانات الدخول الافتراضية

- Password: من Firebase Authentication
